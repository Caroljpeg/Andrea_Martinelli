gridWizard is an open-source tool developed to generate and manage grid systems within Adobe InDesign. for any request / consideration / report or if you just want to share what you used it for, feel free to reach me at a.martinelli0601@gmail.com

v1.0 offers 5 building methods (looking to add some others soon !!). You can check the source code for all of them in the functions directory inside the github repository

1 ― Everything Everywhere All At Page Ratio

generate a layout grid wherein everything is related to the aspect ratio of the page.

2 ― I Kissed A Square And I Liked It

generate a square-based layout system.

3 ― Gutenbergify

parametrically recreate the grid developed by Gutenberg for his 42-line bible.

4 ― Fibonacci Dreams

use the Fibonacci sequence to produce a mathematically harmonious layout grid.

5 ― Pure Chaos

just some pseudo-random chaos.
